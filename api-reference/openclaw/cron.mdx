---
title: "List Cron Jobs"
description: "List all scheduled cron jobs"
api: "GET /api/v1/openclaw/cron"
---

Returns a list of all scheduled cron jobs configured in the OpenClaw gateway, including their schedules, status, and execution history.

<ParamField header="Authorization" type="string" required>
  Bearer token for API authentication. Use `Bearer YOUR_API_KEY`.
</ParamField>

<RequestExample>

```bash cURL
curl --request GET \
  --url https://alfred-acme.tailnet.ts.net:3100/api/v1/openclaw/cron \
  --header 'Authorization: Bearer YOUR_API_KEY'
```

</RequestExample>
