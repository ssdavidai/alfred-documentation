---
title: "List Cron Jobs"
description: "List scheduled cron jobs"
api: "GET /api/v1/openclaw/cron"
---

List all scheduled cron jobs configured on your instance's agent gateway. Cron jobs automate recurring agent tasks like periodic vault sweeps or maintenance runs.

The response is returned as structured JSON when available, or as raw text in a `raw` field.

<RequestExample>

```bash cURL
curl -X GET /api/v1/openclaw/cron \
  -H "Authorization: Bearer alf_your_key_here"
```

</RequestExample>

<ResponseExample>

```json Structured Response
{
  "...": "Structure depends on gateway cron output"
}
```

```json Raw Response
{
  "raw": "No cron jobs configured"
}
```

</ResponseExample>

<Note>
  The response format depends on the gateway's output. If the gateway returns valid JSON, it is passed through as-is. Otherwise, the raw text is returned in a `raw` field.
</Note>
