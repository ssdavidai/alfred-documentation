---
title: "List Plugins"
description: "List installed plugins"
api: "GET /api/v1/openclaw/plugins"
---

List all plugins and extensions installed on your instance's agent gateway. Plugins extend the gateway's capabilities with additional integrations and features.

The response is returned as structured JSON when available, or as raw text in a `raw` field.

<RequestExample>

```bash cURL
curl -X GET /api/v1/openclaw/plugins \
  -H "Authorization: Bearer alf_your_key_here"
```

</RequestExample>

<ResponseExample>

```json Structured Response
{
  "...": "Structure depends on gateway plugins output"
}
```

```json Raw Response
{
  "raw": "No plugins installed"
}
```

</ResponseExample>

<Note>
  The response format depends on the gateway's output. If the gateway returns valid JSON, it is passed through as-is. Otherwise, the raw text is returned in a `raw` field.
</Note>
