---
title: "List Plugins"
description: "List all installed extensions and plugins"
api: "GET /api/v1/openclaw/plugins"
---

Returns a list of all extensions and plugins installed in the OpenClaw gateway, including their status, version, and capabilities.

<ParamField header="Authorization" type="string" required>
  Bearer token for API authentication. Use `Bearer YOUR_API_KEY`.
</ParamField>

<RequestExample>

```bash cURL
curl --request GET \
  --url https://alfred-acme.tailnet.ts.net:3100/api/v1/openclaw/plugins \
  --header 'Authorization: Bearer YOUR_API_KEY'
```

</RequestExample>
