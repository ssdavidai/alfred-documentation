---
title: "List Skills"
description: "List available agent skills"
api: "GET /api/v1/openclaw/skills"
---

List all agent skills available on your instance. Skills define what your AI agents can do -- for example, `vault-curator`, `vault-janitor`, and `vault-distiller` are the built-in Alfred skills.

The response is returned as structured JSON when available, or as raw text in a `raw` field.

<RequestExample>

```bash cURL
curl -X GET /api/v1/openclaw/skills \
  -H "Authorization: Bearer alf_your_key_here"
```

</RequestExample>

<ResponseExample>

```json Structured Response
{
  "...": "Structure depends on gateway skills output"
}
```

```json Raw Response
{
  "raw": "vault-curator, vault-janitor, vault-distiller"
}
```

</ResponseExample>

<Note>
  The response format depends on the gateway's output. If the gateway returns valid JSON, it is passed through as-is. Otherwise, the raw text is returned in a `raw` field.
</Note>
