---
title: "List Skills"
description: "List all available OpenClaw agent skills"
api: "GET /api/v1/openclaw/skills"
---

Returns a list of all available skills that can be used by OpenClaw agents, including their definitions, parameters, and descriptions.

<ParamField header="Authorization" type="string" required>
  Bearer token for API authentication. Use `Bearer YOUR_API_KEY`.
</ParamField>

<RequestExample>

```bash cURL
curl --request GET \
  --url https://alfred-acme.tailnet.ts.net:3100/api/v1/openclaw/skills \
  --header 'Authorization: Bearer YOUR_API_KEY'
```

</RequestExample>
