---
title: "List Secrets"
description: "List configured secrets"
api: "GET /api/v1/openclaw/secrets"
---

List all secret providers configured on your instance's agent gateway. Secret values are always masked -- only the key names and redacted values are returned.

The response is returned as structured JSON when available, or as raw text in a `raw` field.

<RequestExample>

```bash cURL
curl -X GET /api/v1/openclaw/secrets \
  -H "Authorization: Bearer alf_your_key_here"
```

</RequestExample>

<ResponseExample>

```json Structured Response
{
  "...": "Structure depends on gateway secrets output (values are masked)"
}
```

```json Raw Response
{
  "raw": "OPENROUTER_API_KEY=sk-or-v1-***\nHETZNER_S3_ACCESS_KEY=***"
}
```

</ResponseExample>

<Note>
  Secret values are always masked before being returned. You will only see key names and redacted placeholders, never actual secret values.
</Note>
