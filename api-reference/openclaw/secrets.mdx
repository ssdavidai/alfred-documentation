---
title: "List Secrets"
description: "List configured secret providers (values are masked)"
api: "GET /api/v1/openclaw/secrets"
---

Returns a list of all configured secret providers in the OpenClaw gateway. Secret values are masked for security, only metadata is exposed.

<ParamField header="Authorization" type="string" required>
  Bearer token for API authentication. Use `Bearer YOUR_API_KEY`.
</ParamField>

<RequestExample>

```bash cURL
curl --request GET \
  --url https://alfred-acme.tailnet.ts.net:3100/api/v1/openclaw/secrets \
  --header 'Authorization: Bearer YOUR_API_KEY'
```

</RequestExample>
