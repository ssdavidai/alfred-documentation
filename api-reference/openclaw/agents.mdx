---
title: "List Agents"
description: "List registered agents"
api: "GET /api/v1/openclaw/agents"
---

List all agents registered with your instance's agent gateway. Agents are the AI backends that execute skills on behalf of Alfred tools like Curator, Janitor, Distiller, and Surveyor.

The response is returned as structured JSON when available, or as raw text in a `raw` field.

<RequestExample>

```bash cURL
curl -X GET /api/v1/openclaw/agents \
  -H "Authorization: Bearer alf_your_key_here"
```

</RequestExample>

<ResponseExample>

```json Structured Response
{
  "...": "Structure depends on gateway agents output"
}
```

```json Raw Response
{
  "raw": "No agents configured"
}
```

</ResponseExample>

<Note>
  The response format depends on the gateway's output. If the gateway returns valid JSON, it is passed through as-is. Otherwise, the raw text is returned in a `raw` field.
</Note>
