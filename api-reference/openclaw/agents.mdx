---
title: "List Agents"
description: "List all managed OpenClaw agents"
api: "GET /api/v1/openclaw/agents"
---

Returns a list of all agents managed by the OpenClaw gateway, including their configuration, status, and activity metrics.

<ParamField header="Authorization" type="string" required>
  Bearer token for API authentication. Use `Bearer YOUR_API_KEY`.
</ParamField>

<RequestExample>

```bash cURL
curl --request GET \
  --url https://alfred-acme.tailnet.ts.net:3100/api/v1/openclaw/agents \
  --header 'Authorization: Bearer YOUR_API_KEY'
```

</RequestExample>
