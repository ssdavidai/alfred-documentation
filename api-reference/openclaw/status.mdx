---
title: "Gateway Status"
description: "Get detailed OpenClaw gateway status information"
api: "GET /api/v1/openclaw/status"
---

Returns detailed status information about the OpenClaw gateway, including configuration, active connections, and runtime metrics.

<ParamField header="Authorization" type="string" required>
  Bearer token for API authentication. Use `Bearer YOUR_API_KEY`.
</ParamField>

<RequestExample>

```bash cURL
curl --request GET \
  --url https://alfred-acme.tailnet.ts.net:3100/api/v1/openclaw/status \
  --header 'Authorization: Bearer YOUR_API_KEY'
```

</RequestExample>
