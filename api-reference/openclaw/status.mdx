---
title: "Gateway Status"
description: "Get agent gateway status"
api: "GET /api/v1/openclaw/status"
---

Get detailed status information about your instance's agent gateway, including uptime, version, and current configuration.

The response is returned as structured JSON when available, or as raw text in a `raw` field.

<RequestExample>

```bash cURL
curl -X GET /api/v1/openclaw/status \
  -H "Authorization: Bearer alf_your_key_here"
```

</RequestExample>

<ResponseExample>

```json Structured Response
{
  "...": "Structure depends on gateway status output"
}
```

```json Raw Response
{
  "raw": "Gateway status: running"
}
```

</ResponseExample>

<Note>
  The response format depends on the gateway's output. If the gateway returns valid JSON, it is passed through as-is. Otherwise, the raw text is returned in a `raw` field.
</Note>
