---
title: "Security Audit"
description: "Audit OpenClaw gateway security configuration"
api: "GET /api/v1/openclaw/security"
---

Performs a security audit of the OpenClaw gateway configuration and returns a detailed security report with findings and recommendations.

<ParamField header="Authorization" type="string" required>
  Bearer token for API authentication. Use `Bearer YOUR_API_KEY`.
</ParamField>

<RequestExample>

```bash cURL
curl --request GET \
  --url https://alfred-acme.tailnet.ts.net:3100/api/v1/openclaw/security \
  --header 'Authorization: Bearer YOUR_API_KEY'
```

</RequestExample>
