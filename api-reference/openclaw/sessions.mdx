---
title: "List Sessions"
description: "List all conversation sessions"
api: "GET /api/v1/openclaw/sessions"
---

Returns a list of all conversation sessions managed by OpenClaw, including active and completed sessions.

<ParamField header="Authorization" type="string" required>
  Bearer token for API authentication. Use `Bearer YOUR_API_KEY`.
</ParamField>

<RequestExample>

```bash cURL
curl --request GET \
  --url https://alfred-acme.tailnet.ts.net:3100/api/v1/openclaw/sessions \
  --header 'Authorization: Bearer YOUR_API_KEY'
```

</RequestExample>
