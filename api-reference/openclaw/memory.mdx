---
title: "Search Agent Memory"
description: "Vector search over agent memory and conversation history"
api: "GET /api/v1/openclaw/memory"
---

Performs semantic vector search over agent memory and conversation history using embeddings. Returns relevant memory entries ranked by similarity.

<ParamField header="Authorization" type="string" required>
  Bearer token for API authentication. Use `Bearer YOUR_API_KEY`.
</ParamField>

<ParamField query="query" type="string" required>
  The search query text. Will be embedded and used for semantic similarity search.
</ParamField>

<ParamField query="limit" type="number" default={10}>
  Maximum number of results to return.
</ParamField>

<ParamField query="threshold" type="number" default={0.7}>
  Minimum similarity score threshold (0.0 to 1.0). Only results above this threshold are returned.
</ParamField>

<ParamField query="agent_id" type="string">
  Optional filter to search only within a specific agent's memory.
</ParamField>

<RequestExample>

```bash cURL
curl --request GET \
  --url 'https://alfred-acme.tailnet.ts.net:3100/api/v1/openclaw/memory?query=vault%20organization%20patterns&limit=5&threshold=0.75' \
  --header 'Authorization: Bearer YOUR_API_KEY'
```

</RequestExample>
