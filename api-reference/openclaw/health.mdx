---
title: "Gateway Health"
description: "Check OpenClaw gateway health status"
api: "GET /api/v1/openclaw/health"
---

Returns the current health status of the OpenClaw gateway, including uptime, system resources, and service availability.

<ParamField header="Authorization" type="string" required>
  Bearer token for API authentication. Use `Bearer YOUR_API_KEY`.
</ParamField>

<RequestExample>

```bash cURL
curl --request GET \
  --url https://alfred-acme.tailnet.ts.net:3100/api/v1/openclaw/health \
  --header 'Authorization: Bearer YOUR_API_KEY'
```

</RequestExample>
