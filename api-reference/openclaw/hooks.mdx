---
title: "List Hooks"
description: "List all registered event hooks"
api: "GET /api/v1/openclaw/hooks"
---

Returns a list of all event hooks registered in the OpenClaw gateway. Hooks are triggered on specific events and can execute custom actions.

<ParamField header="Authorization" type="string" required>
  Bearer token for API authentication. Use `Bearer YOUR_API_KEY`.
</ParamField>

<RequestExample>

```bash cURL
curl --request GET \
  --url https://alfred-acme.tailnet.ts.net:3100/api/v1/openclaw/hooks \
  --header 'Authorization: Bearer YOUR_API_KEY'
```

</RequestExample>
