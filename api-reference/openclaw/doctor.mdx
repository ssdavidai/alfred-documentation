---
title: "Run Diagnostics"
description: "Run health checks and auto-fixes for OpenClaw gateway"
api: "POST /api/v1/openclaw/doctor"
---

Runs comprehensive health checks across the OpenClaw gateway and attempts to auto-fix any detected issues. Returns a detailed diagnostic report.

<ParamField header="Authorization" type="string" required>
  Bearer token for API authentication. Use `Bearer YOUR_API_KEY`.
</ParamField>

<RequestExample>

```bash cURL
curl --request POST \
  --url https://alfred-acme.tailnet.ts.net:3100/api/v1/openclaw/doctor \
  --header 'Authorization: Bearer YOUR_API_KEY'
```

</RequestExample>
