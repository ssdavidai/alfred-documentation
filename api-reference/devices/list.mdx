---
title: "List Devices"
description: "Retrieve all pending and paired devices"
api: "GET /api/v1/devices"
---

Returns a list of all devices associated with your Alfred instance, including both pending pairing requests and already-paired devices.

<RequestExample>

```bash cURL
curl -X GET /api/v1/devices \
  -H "Authorization: Bearer alf_your_key_here"
```

</RequestExample>

<ResponseExample>

```json Response
{
  "devices": []
}
```

</ResponseExample>
