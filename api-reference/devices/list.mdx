---
title: "List Devices"
description: "Retrieve all pending and paired devices in the OpenClaw gateway"
api: "GET /api/v1/devices"
---

Returns a list of all devices associated with the OpenClaw gateway by running `openclaw devices list --json` inside the container. The JSON output is parsed and returned directly.

<Note>
  Unlike most other endpoints, this endpoint does not have a raw text fallback. The `openclaw devices list --json` output must be valid JSON. If parsing fails, the API will return a 500 error.
</Note>

<RequestExample>

```bash cURL
curl -X GET https://app.alfred.black/api/v1/devices \
  -H "Authorization: Bearer YOUR_API_KEY"
```

</RequestExample>

<ResponseExample>

```json Response
{
  "...": "Structure depends on openclaw devices list --json output"
}
```

</ResponseExample>

<Note>
  The response structure is parsed directly from `openclaw devices list --json` output. The exact fields depend on the OpenClaw version.
</Note>
