---
title: "List Devices"
description: "Retrieve all pending and paired devices in the OpenClaw gateway"
api: "GET /api/v1/devices"
---

Returns a list of all pending and paired devices associated with the OpenClaw gateway. The response is structured JSON.

<Note>
  Unlike most other endpoints, this endpoint always returns structured JSON. If the underlying operation fails, the API will return a 500 error.
</Note>

<RequestExample>

```bash cURL
curl -X GET https://app.alfred.black/api/v1/devices \
  -H "Authorization: Bearer YOUR_API_KEY"
```

</RequestExample>

<ResponseExample>

```json Response
{
  "pending": [],
  "paired": []
}
```

</ResponseExample>

<Note>
  The response typically includes `pending` and `paired` arrays. The exact fields within each device object depend on the OpenClaw version.
</Note>
