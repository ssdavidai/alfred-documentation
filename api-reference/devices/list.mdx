---
title: "List Devices"
description: "Retrieve all pending and paired devices in the OpenClaw gateway"
api: "GET /api/v1/devices"
---

Returns a list of all devices associated with the OpenClaw gateway, separated into pending approval requests and paired devices.

<ParamField header="Authorization" type="string" required>
  Bearer token for API authentication. Use `Bearer YOUR_API_KEY`.
</ParamField>

<RequestExample>

```bash cURL
curl --request GET \
  --url https://alfred-acme.tailnet.ts.net:3100/api/v1/devices \
  --header 'Authorization: Bearer YOUR_API_KEY'
```

</RequestExample>
