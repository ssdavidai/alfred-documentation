---
title: "Remove a device"
description: "Delete a paired device and all of its credentials."
api: "DELETE /api/v1/devices/{deviceId}"
---

Permanently removes a paired device. All tokens associated with the device are invalidated and the entry is dropped from the inventory. Use this when Sir retires a phone, decommissions a workstation, or cleans up a forgotten CLI session.

## Path parameters

<ParamField path="deviceId" type="string" required>
  The `device_id` from `GET /api/v1/devices` (under `devices`).
</ParamField>

## Response

<ResponseField name="message" type="string">
  Confirmation string from the OpenClaw CLI (e.g. `"Device removed"`).
</ResponseField>

<RequestExample>

```bash cURL
curl -X DELETE https://alfred.black/api/v1/devices/phn-7f2c \
  -H "Authorization: Bearer alf_your_key_here"
```

</RequestExample>

<ResponseExample>

```json 200 OK
{
  "message": "Device removed"
}
```

</ResponseExample>

<Warning>
  Removal is destructive and immediate. If the device tries to reconnect, it will need to repeat the full pairing flow.
</Warning>
