---
title: "Remove Device"
description: "Remove a paired device from the OpenClaw gateway"
api: "DELETE /api/v1/devices/{deviceId}"
---

Removes a paired device from the OpenClaw gateway, revoking its access and invalidating any existing tokens.

<ParamField path="deviceId" type="string" required>
  The unique identifier of the paired device to remove.
</ParamField>

<ParamField header="Authorization" type="string" required>
  Bearer token for API authentication. Use `Bearer YOUR_API_KEY`.
</ParamField>

<RequestExample>

```bash cURL
curl --request DELETE \
  --url https://alfred-acme.tailnet.ts.net:3100/api/v1/devices/dev_a1b2c3d4e5 \
  --header 'Authorization: Bearer YOUR_API_KEY'
```

</RequestExample>

<ResponseExample>

```json Response
{
  "message": "Device removed successfully",
  "deviceId": "dev_a1b2c3d4e5"
}
```

</ResponseExample>
