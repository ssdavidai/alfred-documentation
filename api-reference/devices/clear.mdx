---
title: "Clear Devices"
description: "Bulk clear pending or all devices"
api: "POST /api/v1/devices/clear"
---

Clears multiple devices at once. Can be used to clear only pending requests or all devices (both pending and paired).

<ParamField body="pending_only" type="boolean" optional>
  If true, only clears pending device requests. If omitted or false, clears all devices.
</ParamField>

<RequestExample>

```bash Clear Pending Only
curl -X POST https://app.alfred.black/api/v1/devices/clear \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "pending_only": true
  }'
```

```bash Clear All
curl -X POST https://app.alfred.black/api/v1/devices/clear \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json"
```

</RequestExample>

<ResponseExample>

```json Response
{
  "message": "Devices cleared"
}
```

</ResponseExample>

<ResponseField name="message" type="string">
  Human-readable status message confirming devices have been cleared.
</ResponseField>
