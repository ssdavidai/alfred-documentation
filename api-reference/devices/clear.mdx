---
title: "Clear Devices"
description: "Bulk clear pending or all devices"
api: "POST /api/v1/devices/clear"
---

Clears multiple devices at once. Can be used to clear all pending requests or reset all devices (both pending and paired).

<ParamField header="Authorization" type="string" required>
  Bearer token for API authentication. Use `Bearer YOUR_API_KEY`.
</ParamField>

<ParamField body="pending_only" type="boolean" default={true}>
  If true, only clear pending device requests. If false, clear both pending and paired devices.
</ParamField>

<RequestExample>

```bash cURL
curl --request POST \
  --url https://alfred-acme.tailnet.ts.net:3100/api/v1/devices/clear \
  --header 'Authorization: Bearer YOUR_API_KEY' \
  --header 'Content-Type: application/json' \
  --data '{
    "pending_only": true
  }'
```

</RequestExample>

<ResponseExample>

```json Response
{
  "message": "Cleared 3 pending device requests",
  "pending_cleared": 3,
  "paired_cleared": 0
}
```

</ResponseExample>
