---
title: "Clear Devices"
description: "Bulk clear pending or all devices"
api: "POST /api/v1/devices/clear"
---

Clears multiple devices at once. You can clear only pending pairing requests or remove all devices (both pending and paired).

<ParamField body="pending_only" type="boolean" optional>
  If true, only clears pending device requests. If omitted or false, clears all devices.
</ParamField>

<RequestExample>

```bash Clear Pending Only
curl -X POST /api/v1/devices/clear \
  -H "Authorization: Bearer alf_your_key_here" \
  -H "Content-Type: application/json" \
  -d '{
    "pending_only": true
  }'
```

```bash Clear All
curl -X POST /api/v1/devices/clear \
  -H "Authorization: Bearer alf_your_key_here" \
  -H "Content-Type: application/json"
```

</RequestExample>

<ResponseExample>

```json Response
{
  "message": "Devices cleared"
}
```

</ResponseExample>

<ResponseField name="message" type="string">
  A status message confirming the devices were cleared.
</ResponseField>
