---
title: "List Workflows"
description: "Retrieve a list of workflows with optional filtering"
api: "GET /api/v1/workflows"
---

Returns a list of workflows in the Temporal namespace. Runs `temporal workflow list --output json` inside the temporal container; the JSONL output is parsed into an array. Supports optional filtering using Temporal query syntax.

<ParamField query="query" type="string" optional>
  Temporal visibility query string for filtering workflows. Uses Temporal's SQL-like query syntax. Examples:
  - `ExecutionStatus='Running'`
  - `WorkflowType='MyWorkflow' AND ExecutionStatus='Completed'`
  - `StartTime > '2026-01-01T00:00:00Z'`
</ParamField>

<RequestExample>
```bash cURL
# List all workflows
curl -X GET \
  -H "Authorization: Bearer YOUR_API_KEY" \
  https://app.alfred.black/api/v1/workflows

# List running workflows
curl -X GET \
  -H "Authorization: Bearer YOUR_API_KEY" \
  "https://app.alfred.black/api/v1/workflows?query=ExecutionStatus%3D%27Running%27"
```
</RequestExample>
