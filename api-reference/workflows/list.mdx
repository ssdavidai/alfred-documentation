---
title: "List Workflows"
description: "Retrieve a list of workflows with optional filtering"
api: "GET https://alfred-acme.tailnet.ts.net:3100/api/v1/workflows"
---

Returns a list of workflows in the Temporal namespace. Supports optional filtering using Temporal query syntax.

<ParamField query="query" type="string" optional>
  Temporal query string for filtering workflows. Examples:
  - `ExecutionStatus='Running'`
  - `WorkflowType='CuratorWorkflow' AND ExecutionStatus='Completed'`
  - `StartTime > '2026-01-01T00:00:00Z'`
</ParamField>

<RequestExample>
```bash cURL
# List all workflows
curl -X GET \
  -H "Authorization: Bearer YOUR_API_KEY" \
  https://alfred-acme.tailnet.ts.net:3100/api/v1/workflows

# List running workflows
curl -X GET \
  -H "Authorization: Bearer YOUR_API_KEY" \
  "https://alfred-acme.tailnet.ts.net:3100/api/v1/workflows?query=ExecutionStatus%3D%27Running%27"
```
</RequestExample>
