---
title: "Get Workflow History"
description: "Retrieve the complete event history for a workflow execution"
api: "GET /api/v1/workflows/{wfId}/history"
---

Returns the complete event history for a workflow execution. Runs `temporal workflow show --output json` and parses the JSONL output into an array.

<ParamField path="wfId" type="string" required>
  The workflow ID to retrieve history for
</ParamField>

<ParamField query="run_id" type="string" optional>
  Specific run ID to retrieve history for. If omitted, returns history for the most recent run.
</ParamField>

<RequestExample>
```bash cURL
curl -X GET \
  -H "Authorization: Bearer YOUR_API_KEY" \
  https://app.alfred.black/api/v1/workflows/curator-sweep-20260226-143022/history
```
</RequestExample>

<ResponseExample>
```json 200
[
  {
    "eventId": "1",
    "eventType": "EVENT_TYPE_WORKFLOW_EXECUTION_STARTED",
    "eventTime": "2026-02-26T14:30:22.154Z",
    "workflowExecutionStartedEventAttributes": {
      "workflowType": {
        "name": "CuratorWorkflow"
      },
      "taskQueue": {
        "name": "alfred-curator"
      }
    }
  },
  {
    "eventId": "2",
    "eventType": "EVENT_TYPE_WORKFLOW_TASK_SCHEDULED",
    "eventTime": "2026-02-26T14:30:22.160Z"
  }
]
```
</ResponseExample>

<Note>
  The response is a JSON array of history events parsed from the Temporal CLI's JSONL output. The exact event types and attribute structures are determined by the Temporal CLI version.
</Note>
