---
title: "Get Workflow History"
description: "Retrieve the complete event history for a workflow execution"
api: "GET https://alfred-acme.tailnet.ts.net:3100/api/v1/workflows/{wfId}/history"
---

Returns the complete event history for a workflow execution. History events include workflow start, activity scheduling, task completion, signals, and more.

<ParamField path="wfId" type="string" required>
  The workflow ID to retrieve history for
</ParamField>

<ParamField query="run_id" type="string" optional>
  Specific run ID to retrieve history for. If omitted, returns history for the most recent run.
</ParamField>

<RequestExample>
```bash cURL
curl -X GET \
  -H "Authorization: Bearer YOUR_API_KEY" \
  https://alfred-acme.tailnet.ts.net:3100/api/v1/workflows/curator-sweep-20260226-143022/history
```
</RequestExample>
