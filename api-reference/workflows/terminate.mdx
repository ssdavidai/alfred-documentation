---
title: "Terminate Workflow"
description: "Forcefully terminate a running workflow"
api: "POST /api/v1/workflows/{wfId}/terminate"
---

Immediately terminates a workflow execution. This is a hard stop -- the workflow cannot be resumed and will not execute cleanup logic.

<ParamField path="wfId" type="string" required>
  The workflow ID to terminate
</ParamField>

<ParamField body="reason" type="string" optional>
  Human-readable reason for termination. Will be recorded in workflow history.
</ParamField>

<ParamField body="run_id" type="string" optional>
  Specific run ID to terminate. If omitted, terminates the most recent run.
</ParamField>

<RequestExample>
```bash cURL
curl -X POST \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "reason": "Manual termination - inbox processing stuck"
  }' \
  https://app.alfred.black/api/v1/workflows/curator-sweep-20260226-143022/terminate
```
</RequestExample>

<ResponseExample>
```json 200
{
  "message": "Workflow terminated"
}
```
</ResponseExample>
