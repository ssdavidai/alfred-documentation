---
title: "Cancel Workflow"
description: "Request graceful cancellation of a workflow"
api: "POST /api/v1/workflows/{wfId}/cancel"
---

Requests graceful cancellation of a workflow. Unlike terminate, this allows the workflow to execute cleanup logic before stopping.

<ParamField path="wfId" type="string" required>
  The workflow ID to cancel
</ParamField>

<ParamField body="run_id" type="string" optional>
  Specific run ID to cancel. If omitted, cancels the most recent run.
</ParamField>

<RequestExample>
```bash cURL
curl -X POST \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{}' \
  https://app.alfred.black/api/v1/workflows/curator-sweep-20260226-143022/cancel
```
</RequestExample>

<ResponseExample>
```json 200
{
  "message": "Workflow cancelled"
}
```
</ResponseExample>
