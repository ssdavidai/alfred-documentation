api: "GET /api/v1/integrations/catalog"
---

## Overview

Returns the full Composio toolkit catalogue with pagination, search, and category filtering. The result is cached for one hour.

## Query Parameters

<ParamField query="search" type="string" optional>
  Case-insensitive substring match against the toolkit slug, name, or description.
</ParamField>

<ParamField query="category" type="string" optional>
  Filter to a single category (e.g. `productivity`, `crm`, `dev_tools`). Pass `all` or omit for no filter.
</ParamField>

## Response

<ResponseField name="toolkits" type="array">
  Filtered toolkit list. Each entry includes `slug`, `name`, `description`, `category`, `logo`, and `auth_schemes`.
</ResponseField>

<ResponseField name="categories" type="array">
  All distinct categories present in the unfiltered catalogue.
</ResponseField>

<ResponseField name="count" type="number">
  Filtered count.
</ResponseField>

<ResponseField name="total" type="number">
  Unfiltered total.
</ResponseField>

## Request Example

<RequestExample>

```bash cURL
curl -H "Authorization: Bearer $AAS_API_KEY" \
  "https://alfred.black/api/v1/integrations/catalog?search=notion"
```

</RequestExample>
