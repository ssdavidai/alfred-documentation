---
title: "Pause Schedule"
description: "Pause a workflow schedule"
api: "POST /api/v1/schedules/{schId}/pause"
---

Pause a workflow schedule to temporarily stop new executions from being started. Currently running workflows are not affected. Use [unpause](/api-reference/schedules/unpause) to resume the schedule.

<ParamField path="schId" type="string" required>
  The schedule ID to pause
</ParamField>

<ParamField body="reason" type="string" optional>
  Human-readable reason for pausing. Recorded on the schedule for reference.
</ParamField>

<RequestExample>
```bash cURL
curl -X POST \
  -H "Authorization: Bearer alf_your_key_here" \
  -H "Content-Type: application/json" \
  -d '{
    "reason": "Paused during vault migration"
  }' \
  /api/v1/schedules/curator-hourly/pause
```
</RequestExample>

<ResponseExample>
```json 200
{
  "message": "Schedule paused"
}
```
</ResponseExample>
