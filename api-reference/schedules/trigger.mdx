---
title: "Trigger Schedule"
description: "Manually trigger an immediate workflow execution from a schedule"
api: "POST https://alfred-acme.tailnet.ts.net:3100/api/v1/schedules/{schId}/trigger"
---

Immediately triggers a workflow execution from a schedule, bypassing the normal cron timing. The schedule's overlap policy is respected.

<ParamField path="schId" type="string" required>
  The schedule ID to trigger
</ParamField>

<RequestExample>
```bash cURL
curl -X POST \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{}' \
  https://alfred-acme.tailnet.ts.net:3100/api/v1/schedules/curator-hourly/trigger
```
</RequestExample>

<ResponseExample>
```json 200 OK
{
  "success": true,
  "scheduleId": "curator-hourly",
  "triggered": true,
  "workflowId": "curator-hourly-manual-2026-02-26T14:45:33Z",
  "runId": "c3d4e5f6-a7b8-9012-cdef-123456789012",
  "startTime": "2026-02-26T14:45:33.891Z"
}
```
</ResponseExample>
