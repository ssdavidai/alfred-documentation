---
title: "Unpause Schedule"
description: "Resume a paused workflow schedule"
api: "POST /api/v1/schedules/{schId}/unpause"
---

Resumes a paused workflow schedule. The schedule will begin executing according to its cron expression.

<ParamField path="schId" type="string" required>
  The schedule ID to unpause
</ParamField>

<ParamField body="reason" type="string" optional>
  Human-readable reason for unpausing. Passed to the Temporal CLI.
</ParamField>

<RequestExample>
```bash cURL
curl -X POST \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "reason": "Vault migration complete"
  }' \
  https://app.alfred.black/api/v1/schedules/curator-hourly/unpause
```
</RequestExample>

<ResponseExample>
```json 200
{
  "message": "Schedule unpaused"
}
```
</ResponseExample>
