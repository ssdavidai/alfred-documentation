---
title: "Describe Schedule"
description: "Get detailed information about a specific schedule"
api: "GET https://alfred-acme.tailnet.ts.net:3100/api/v1/schedules/{schId}"
---

Returns comprehensive details about a workflow schedule, including configuration, state, and recent execution history.

<ParamField path="schId" type="string" required>
  The schedule ID to describe
</ParamField>

<RequestExample>
```bash cURL
curl -X GET \
  -H "Authorization: Bearer YOUR_API_KEY" \
  https://alfred-acme.tailnet.ts.net:3100/api/v1/schedules/curator-hourly
```
</RequestExample>
