---
title: "List Schedules"
description: "Retrieve all workflow schedules"
api: "GET https://alfred-acme.tailnet.ts.net:3100/api/v1/schedules"
---

Returns a list of all workflow schedules configured in the Temporal namespace.

<RequestExample>
```bash cURL
curl -X GET \
  -H "Authorization: Bearer YOUR_API_KEY" \
  https://alfred-acme.tailnet.ts.net:3100/api/v1/schedules
```
</RequestExample>
