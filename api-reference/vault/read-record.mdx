---
title: "Read Record"
description: "Read the full content of a vault record."
api: "GET /api/v1/vault/records/{path}"
---

## Overview

Returns the full content of a vault record including its metadata and body.

## Path Parameters

<ParamField path="path" type="string" required>
  Relative path to the record (e.g., `person/Alice Chen.md`). URL-encode spaces and special characters.
</ParamField>

## Response

<ResponseField name="raw" type="string">
  The full record content (metadata + body) as text. If the output is structured JSON, it is returned directly instead.
</ResponseField>

## Request Example

<RequestExample>

```bash cURL
curl -X GET \
  "/api/v1/vault/records/person/Alice%20Chen.md" \
  -H "Authorization: Bearer alf_your_key_here"
```

```javascript JavaScript
const recordPath = 'person/Alice Chen.md';
const encodedPath = encodeURIComponent(recordPath);

const response = await fetch(`/api/v1/vault/records/${encodedPath}`, {
  method: 'GET',
  headers: {
    'Authorization': 'Bearer alf_your_key_here'
  }
});

const record = await response.json();
console.log(record);
```

</RequestExample>
