---
title: "Read Record"
description: "Read a specific vault record by path"
api: "GET /api/v1/vault/records/{path}"
---

## Overview

Returns the full contents of a vault record, including parsed frontmatter (YAML metadata) and markdown body.

## Path Parameters

<ParamField path="path" type="string" required>
  Relative path to the record (e.g., `person/Alice.md`, `project/ML Pipeline.md`). URL-encode if the path contains spaces or special characters.
</ParamField>

## Response

<ResponseField name="path" type="string">
  Relative path to the record
</ResponseField>

<ResponseField name="frontmatter" type="object">
  Parsed YAML frontmatter as key-value pairs
  <ResponseField name="type" type="string">
    Record type
  </ResponseField>
  <ResponseField name="status" type="string">
    Current status
  </ResponseField>
  <ResponseField name="created" type="string">
    ISO 8601 creation timestamp
  </ResponseField>
  <ResponseField name="updated" type="string">
    ISO 8601 last update timestamp
  </ResponseField>
</ResponseField>

<ResponseField name="body" type="string">
  Markdown body content (everything after the frontmatter block)
</ResponseField>

## Request Example

<RequestExample>

```bash cURL
curl -X GET \
  https://alfred-acme.tailnet.ts.net:3100/api/v1/vault/records/person/Alice%20Chen.md \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```javascript JavaScript
const recordPath = 'person/Alice Chen.md';
const encodedPath = encodeURIComponent(recordPath);

const response = await fetch(`https://alfred-acme.tailnet.ts.net:3100/api/v1/vault/records/${encodedPath}`, {
  method: 'GET',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY'
  }
});

const record = await response.json();
console.log(record);
```

</RequestExample>
