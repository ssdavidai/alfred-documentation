---
title: "Move Record"
description: "Rename or retype a vault record."
api: "POST /api/v1/vault/move"
---

## Overview

Moves a record to a new location — use this to rename a record or change its type. Connections in other records that reference the moved record are updated automatically.

## Body Parameters

<ParamField body="from" type="string" required>
  Current relative path to the record (e.g., `person/Alice.md`).
</ParamField>

<ParamField body="to" type="string" required>
  New relative path for the record (e.g., `person/Alice Chen.md`).
</ParamField>

## Response

<ResponseField name="raw" type="string">
  Confirmation message. If the output is structured JSON, it is returned directly instead.
</ResponseField>

## Request Example

<RequestExample>

```bash cURL
curl -X POST /api/v1/vault/move \
  -H "Authorization: Bearer alf_your_key_here" \
  -H "Content-Type: application/json" \
  -d '{
    "from": "person/Alice.md",
    "to": "person/Alice Chen.md"
  }'
```

```javascript JavaScript
const response = await fetch('/api/v1/vault/move', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer alf_your_key_here',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    from: 'person/Alice.md',
    to: 'person/Alice Chen.md'
  })
});

const result = await response.json();
console.log(result);
```

</RequestExample>

## Response Example

<ResponseExample>

```json 200 OK
{
  "raw": "Moved person/Alice.md to person/Alice Chen.md"
}
```

```json 400 Bad Request
{
  "error": {
    "code": "VALIDATION_ERROR",
    "message": "from and to are required"
  }
}
```

```json 500 Internal Server Error
{
  "error": {
    "code": "EXEC_ERROR",
    "message": "Command failed with exit code 1"
  }
}
```

</ResponseExample>
