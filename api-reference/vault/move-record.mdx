---
title: "Move Record"
description: "Move or rename a vault record"
api: "POST /api/v1/vault/move"
---

## Overview

Moves or renames a vault record. The endpoint runs `alfred vault move <from> <to>` and returns the CLI output. If the Obsidian CLI is available, wikilinks in other vault records referencing the moved record may also be updated.

## Body Parameters

<ParamField body="from" type="string" required>
  Current relative path to the record (e.g., `person/Alice.md`)
</ParamField>

<ParamField body="to" type="string" required>
  New relative path for the record (e.g., `person/Alice Chen.md` or `person/archived/Alice.md`)
</ParamField>

## Response

The response structure depends on the `alfred vault move` CLI output. If the CLI returns valid JSON, it is passed through directly. Otherwise, the raw text output is wrapped in a `{ "raw": "..." }` object.

<ResponseField name="raw" type="string">
  If the CLI output is not valid JSON, the raw text output is returned in this field.
</ResponseField>

## Request Example

<RequestExample>

```bash cURL
curl -X POST \
  https://app.alfred.black/api/v1/vault/move \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "from": "person/Alice.md",
    "to": "person/Alice Chen.md"
  }'
```

```javascript JavaScript
const response = await fetch('https://app.alfred.black/api/v1/vault/move', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    from: 'person/Alice.md',
    to: 'person/Alice Chen.md'
  })
});

const result = await response.json();
console.log(result);
```

</RequestExample>

## Response Example

<ResponseExample>

```json 200 OK (raw text fallback)
{
  "raw": "Moved person/Alice.md to person/Alice Chen.md"
}
```

```json 400 Bad Request
{
  "error": {
    "code": "VALIDATION_ERROR",
    "message": "from and to are required"
  }
}
```

```json 401 Unauthorized
{
  "error": {
    "code": "UNAUTHORIZED",
    "message": "Unauthorized"
  }
}
```

```json 500 Internal Server Error
{
  "error": {
    "code": "EXEC_ERROR",
    "message": "Command failed with exit code 1"
  }
}
```

</ResponseExample>
