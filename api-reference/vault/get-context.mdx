title: "Get Vault Context"
description: "Returns the output of the alfred vault context CLI command"
api: "GET /api/v1/vault/context"
---

## Overview

Returns the output of `alfred vault context`. If the CLI returns valid JSON, it is passed through directly. Otherwise, the raw text output is wrapped in a `{ "raw": "..." }` object.

## Response

The response structure depends on the `alfred vault context` CLI output.

<ResponseField name="raw" type="string">
  If the CLI output is not valid JSON, the raw text output is returned in this field. If the CLI does return valid JSON, the parsed JSON is returned directly instead (structure varies).
</ResponseField>

## Request Example

<RequestExample>

```bash cURL
curl -X GET \
  https://app.alfred.black/api/v1/vault/context \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```javascript JavaScript
const response = await fetch('https://app.alfred.black/api/v1/vault/context', {
  method: 'GET',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY'
  }
});

const context = await response.json();
console.log(context);
```

</RequestExample>

## Response Example

<ResponseExample>

```json 200 OK (raw text fallback)
{
  "raw": "127 records across 18 types.\n\nMost recent activity: 3 tasks updated today, 2 people added this week."
}
```

```json 401 Unauthorized
{
  "error": {
    "code": "UNAUTHORIZED",
    "message": "Unauthorized"
  }
}
```

```json 500 Internal Server Error
{
  "error": {
    "code": "EXEC_ERROR",
    "message": "Command failed with exit code 1"
  }
}
```

</ResponseExample>
