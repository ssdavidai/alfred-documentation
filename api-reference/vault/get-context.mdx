api: "GET /api/v1/vault/context"
---

## Overview

Returns a summary of your vault including record counts by type, recent activity, and overall statistics. Use this to get a quick snapshot of your vault's current state.

## Response

<ResponseField name="raw" type="string">
  Summary of your vault as human-readable text. If the response is structured JSON, it is returned directly instead.
</ResponseField>

## Request Example

<RequestExample>

```bash cURL
curl -X GET /api/v1/vault/context \
  -H "Authorization: Bearer alf_your_key_here"
```

```javascript JavaScript
const response = await fetch('/api/v1/vault/context', {
  method: 'GET',
  headers: {
    'Authorization': 'Bearer alf_your_key_here'
  }
});

const context = await response.json();
console.log(context);
```

</RequestExample>
