---
title: "Edit Record"
description: "Edit an existing vault record"
api: "PATCH /api/v1/vault/records/{path}"
---

## Overview

Modifies an existing vault record. You can set or replace frontmatter fields, append to list fields, append to the markdown body, or replace the body entirely. The endpoint runs `alfred vault edit <path>` and returns the CLI output.

## Path Parameters

<ParamField path="path" type="string" required>
  Relative path to the record (e.g., `person/Alice.md`). URL-encode if the path contains spaces or special characters.
</ParamField>

## Body Parameters

<ParamField body="set" type="object" optional>
  Key-value pairs to set in frontmatter. Each key-value pair is passed as `--set key=value` to the CLI. Replaces existing values for these fields.
</ParamField>

<ParamField body="append" type="object" optional>
  Key-value pairs to append to list fields in frontmatter. Each key-value pair is passed as `--append key=value` to the CLI.
</ParamField>

<ParamField body="body_append" type="string" optional>
  Text to append to the markdown body. Passed as `--body-append` to the CLI.
</ParamField>

<ParamField body="body_stdin" type="string" optional>
  Replace the markdown body with this text. Passed as `--body-stdin` to the CLI.
</ParamField>

<ParamField body="scope" type="string" optional>
  Operation scope for authorization (`curator`, `janitor`, `distiller`). Defaults to no scope restrictions.
</ParamField>

## Response

The response structure depends on the `alfred vault edit` CLI output. If the CLI returns valid JSON, it is passed through directly. Otherwise, the raw text output is wrapped in a `{ "raw": "..." }` object.

<ResponseField name="raw" type="string">
  If the CLI output is not valid JSON, the raw text output is returned in this field.
</ResponseField>

## Request Example

<RequestExample>

```bash cURL
curl -X PATCH \
  https://app.alfred.black/api/v1/vault/records/person/Alice%20Chen.md \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "set": {
      "status": "inactive",
      "updated": "2026-02-26T10:00:00Z"
    },
    "append": {
      "project": "[[project/Infrastructure Audit]]"
    },
    "body_append": "\n## Status Note\n\nMarked inactive as of Feb 26."
  }'
```

```javascript JavaScript
const recordPath = 'person/Alice Chen.md';
const encodedPath = encodeURIComponent(recordPath);

const response = await fetch(`https://app.alfred.black/api/v1/vault/records/${encodedPath}`, {
  method: 'PATCH',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    set: {
      status: 'inactive',
      updated: '2026-02-26T10:00:00Z'
    },
    append: {
      project: '[[project/Infrastructure Audit]]'
    },
    body_append: '\n## Status Note\n\nMarked inactive as of Feb 26.'
  })
});

const result = await response.json();
console.log(result);
```

</RequestExample>
