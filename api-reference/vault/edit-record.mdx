---
title: "Edit Record"
description: "Edit an existing vault record"
api: "PATCH /api/v1/vault/records/{path}"
---

## Overview

Modifies an existing vault record. You can set or replace frontmatter fields, append to list fields, or append to the markdown body.

## Path Parameters

<ParamField path="path" type="string" required>
  Relative path to the record (e.g., `person/Alice.md`). URL-encode if the path contains spaces or special characters.
</ParamField>

## Body Parameters

<ParamField body="set" type="object" optional>
  Key-value pairs to set in frontmatter. Replaces existing values for these fields.
</ParamField>

<ParamField body="append" type="object" optional>
  Key-value pairs to append to list fields in frontmatter. Only works for fields defined as lists in the schema.
</ParamField>

<ParamField body="body_append" type="string" optional>
  Text to append to the markdown body.
</ParamField>

<ParamField body="body_stdin" type="string" optional>
  Completely replace the markdown body with this text.
</ParamField>

<ParamField body="scope" type="string" optional>
  Operation scope for authorization (`curator`, `janitor`, `distiller`). Defaults to no scope restrictions.
</ParamField>

## Response

<ResponseField name="path" type="string">
  Path to the edited record
</ResponseField>

<ResponseField name="updated" type="boolean">
  Always `true` on success
</ResponseField>

## Request Example

<RequestExample>

```bash cURL
curl -X PATCH \
  https://alfred-acme.tailnet.ts.net:3100/api/v1/vault/records/person/Alice%20Chen.md \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "set": {
      "status": "on_leave",
      "updated": "2026-02-26T10:00:00Z"
    },
    "append": {
      "projects": ["[[project/Infrastructure Audit]]"]
    },
    "body_append": "\n## Leave Note\n\nOut Feb 26 - Mar 15 for sabbatical."
  }'
```

```javascript JavaScript
const recordPath = 'person/Alice Chen.md';
const encodedPath = encodeURIComponent(recordPath);

const response = await fetch(`https://alfred-acme.tailnet.ts.net:3100/api/v1/vault/records/${encodedPath}`, {
  method: 'PATCH',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    set: {
      status: 'on_leave',
      updated: '2026-02-26T10:00:00Z'
    },
    append: {
      projects: ['[[project/Infrastructure Audit]]']
    },
    body_append: '\n## Leave Note\n\nOut Feb 26 - Mar 15 for sabbatical.'
  })
});

const result = await response.json();
console.log(result);
```

</RequestExample>

## Response Example

<ResponseExample>

```json 200 OK
{
  "path": "person/Alice Chen.md",
  "updated": true
}
```

```json 404 Not Found
{
  "error": "Record not found: person/Unknown Person.md"
}
```

```json 400 Bad Request
{
  "error": "Cannot append to non-list field 'status'"
}
```

```json 403 Forbidden
{
  "error": "Scope 'distiller' does not allow editing entity types"
}
```

</ResponseExample>
