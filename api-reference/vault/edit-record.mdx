---
title: "Edit Record"
description: "Update an existing vault record's metadata or content."
api: "PATCH /api/v1/vault/records/{path}"
---

## Overview

Updates an existing record's metadata fields and/or content body. You can set or replace fields, append to list fields, or update the content body.

## Path Parameters

<ParamField path="path" type="string" required>
  Relative path to the record (e.g., `person/Alice Chen.md`). URL-encode spaces and special characters.
</ParamField>

## Body Parameters

<ParamField body="set" type="object" optional>
  Key-value pairs to set in the record metadata. Replaces existing values for these fields.
</ParamField>

<ParamField body="append" type="object" optional>
  Key-value pairs to append to list fields in metadata (e.g., adding a tag or project link).
</ParamField>

<ParamField body="body_append" type="string" optional>
  Text to append to the content body.
</ParamField>

<ParamField body="body_stdin" type="string" optional>
  Replace the content body entirely with this text.
</ParamField>

## Response

<ResponseField name="raw" type="string">
  Confirmation message. If the output is structured JSON, it is returned directly instead.
</ResponseField>

## Request Example

<RequestExample>

```bash cURL
curl -X PATCH \
  "/api/v1/vault/records/person/Alice%20Chen.md" \
  -H "Authorization: Bearer alf_your_key_here" \
  -H "Content-Type: application/json" \
  -d '{
    "set": {
      "status": "inactive",
      "updated": "2026-02-26T10:00:00Z"
    },
    "append": {
      "project": "[[project/Infrastructure Audit]]"
    },
    "body_append": "\n## Status Note\n\nMarked inactive as of Feb 26."
  }'
```

```javascript JavaScript
const recordPath = 'person/Alice Chen.md';
const encodedPath = encodeURIComponent(recordPath);

const response = await fetch(`/api/v1/vault/records/${encodedPath}`, {
  method: 'PATCH',
  headers: {
    'Authorization': 'Bearer alf_your_key_here',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    set: {
      status: 'inactive',
      updated: '2026-02-26T10:00:00Z'
    },
    append: {
      project: '[[project/Infrastructure Audit]]'
    },
    body_append: '\n## Status Note\n\nMarked inactive as of Feb 26.'
  })
});

const result = await response.json();
console.log(result);
```

</RequestExample>
