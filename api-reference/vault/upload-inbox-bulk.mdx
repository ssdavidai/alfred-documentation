---
title: "Upload to Inbox (Bulk)"
description: "Upload multiple files to your inbox at once."
api: "POST /api/v1/vault/inbox/bulk"
---

## Overview

Uploads multiple files to your inbox in a single request. The Curator agent will process each file into structured vault records. If any file has an invalid filename or missing content, the entire request fails.

## Body Parameters

<ParamField body="files" type="array" required>
  Array of file objects to upload.
  <ParamField body="filename" type="string" required>
    Name of the file.
  </ParamField>
  <ParamField body="content" type="string" required>
    File contents as a string.
  </ParamField>
</ParamField>

## Response

<ResponseField name="message" type="string">
  Confirmation message.
</ResponseField>

<ResponseField name="filenames" type="array">
  Array of filenames that were written to the inbox.
</ResponseField>

## Request Example

<RequestExample>

```bash cURL
curl -X POST /api/v1/vault/inbox/bulk \
  -H "Authorization: Bearer alf_your_key_here" \
  -H "Content-Type: application/json" \
  -d '{
    "files": [
      {
        "filename": "meeting-notes-2026-02-26.md",
        "content": "# Standup Notes\n\n- Alice: ML pipeline work"
      },
      {
        "filename": "project-idea.md",
        "content": "# New Project Idea\n\nBuild a knowledge graph..."
      },
      {
        "filename": "contact-info.txt",
        "content": "Name: David Lee\nEmail: david@example.com"
      }
    ]
  }'
```

```javascript JavaScript
const response = await fetch('/api/v1/vault/inbox/bulk', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer alf_your_key_here',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    files: [
      { filename: 'meeting-notes-2026-02-26.md', content: '# Standup Notes\n\n...' },
      { filename: 'project-idea.md', content: '# New Project Idea\n\n...' },
      { filename: 'contact-info.txt', content: 'Name: David Lee\n...' }
    ]
  })
});

const result = await response.json();
console.log(result);
```

</RequestExample>

## Response Example

<ResponseExample>

```json 201 Created
{
  "message": "Uploaded 3 files to inbox",
  "filenames": [
    "meeting-notes-2026-02-26.md",
    "project-idea.md",
    "contact-info.txt"
  ]
}
```

```json 400 Bad Request
{
  "error": {
    "code": "VALIDATION_ERROR",
    "message": "files array is required"
  }
}
```

```json 400 Bad Request
{
  "error": {
    "code": "VALIDATION_ERROR",
    "message": "Each file must have filename and content"
  }
}
```

</ResponseExample>
