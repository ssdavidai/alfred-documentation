---
title: "Upload Inbox Files (Bulk)"
description: "Upload multiple files to the vault inbox"
api: "POST /api/v1/vault/inbox/bulk"
---

## Overview

Uploads multiple files to the vault's inbox directory in a single request. Files in the inbox are processed by Alfred's Curator daemon, which converts raw inputs into structured vault records. Each filename is sanitized before writing. If any file has an invalid filename or missing content, the entire request fails.

## Body Parameters

<ParamField body="files" type="array" required>
  Array of file objects to upload
  <ParamField body="filename" type="string" required>
    Name of the file to create in the inbox
  </ParamField>
  <ParamField body="content" type="string" required>
    File contents as a string.
  </ParamField>
</ParamField>

## Response

<ResponseField name="message" type="string">
  Confirmation message (e.g., `"Uploaded 3 files to inbox"`)
</ResponseField>

<ResponseField name="filenames" type="array">
  Array of sanitized filenames that were written to the inbox
</ResponseField>

## Request Example

<RequestExample>

```bash cURL
curl -X POST \
  https://app.alfred.black/api/v1/vault/inbox/bulk \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "files": [
      {
        "filename": "meeting-notes-2026-02-26.md",
        "content": "# Standup Notes\n\n- Alice: ML pipeline work"
      },
      {
        "filename": "project-idea.md",
        "content": "# New Project Idea\n\nBuild a knowledge graph..."
      },
      {
        "filename": "contact-info.txt",
        "content": "Name: David Lee\nEmail: david@example.com"
      }
    ]
  }'
```

```javascript JavaScript
const response = await fetch('https://app.alfred.black/api/v1/vault/inbox/bulk', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    files: [
      {
        filename: 'meeting-notes-2026-02-26.md',
        content: '# Standup Notes\n\n- Alice: ML pipeline work'
      },
      {
        filename: 'project-idea.md',
        content: '# New Project Idea\n\nBuild a knowledge graph...'
      },
      {
        filename: 'contact-info.txt',
        content: 'Name: David Lee\nEmail: david@example.com'
      }
    ]
  })
});

const result = await response.json();
console.log(result);
```

</RequestExample>

## Response Example

<ResponseExample>

```json 201 Created
{
  "message": "Uploaded 3 files to inbox",
  "filenames": [
    "meeting-notes-2026-02-26.md",
    "project-idea.md",
    "contact-info.txt"
  ]
}
```

```json 400 Bad Request
{
  "error": {
    "code": "VALIDATION_ERROR",
    "message": "files array is required"
  }
}
```

```json 400 Bad Request (invalid file entry)
{
  "error": {
    "code": "VALIDATION_ERROR",
    "message": "Each file must have filename and content"
  }
}
```

```json 401 Unauthorized
{
  "error": {
    "code": "UNAUTHORIZED",
    "message": "Unauthorized"
  }
}
```

</ResponseExample>
