---
title: "Upload Inbox Files (Bulk)"
description: "Upload multiple files to the vault inbox"
api: "POST /api/v1/vault/inbox/bulk"
---

## Overview

Uploads multiple files to the vault's inbox directory in a single request. Files in the inbox are processed by Alfred's Curator daemon, which converts raw inputs into structured vault records.

## Body Parameters

<ParamField body="files" type="array" required>
  Array of file objects to upload
  <ParamField body="filename" type="string" required>
    Name of the file to create in the inbox
  </ParamField>
  <ParamField body="content" type="string" required>
    File contents as a string. For binary files, use base64 encoding.
  </ParamField>
</ParamField>

## Response

<ResponseField name="uploaded" type="number">
  Number of files successfully uploaded
</ResponseField>

<ResponseField name="files" type="array">
  Array of uploaded file paths
  <ResponseField name="path" type="string">
    Path to each uploaded file in the inbox
  </ResponseField>
</ResponseField>

<ResponseField name="errors" type="array" optional>
  Array of errors for files that failed to upload (only present if some uploads failed)
  <ResponseField name="filename" type="string">
    Name of the file that failed
  </ResponseField>
  <ResponseField name="error" type="string">
    Error message
  </ResponseField>
</ResponseField>

## Request Example

<RequestExample>

```bash cURL
curl -X POST \
  https://alfred-acme.tailnet.ts.net:3100/api/v1/vault/inbox/bulk \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "files": [
      {
        "filename": "meeting-notes-2026-02-26.md",
        "content": "# Standup Notes\n\n- Alice: ML pipeline work"
      },
      {
        "filename": "project-idea.md",
        "content": "# New Project Idea\n\nBuild a knowledge graph..."
      },
      {
        "filename": "contact-info.txt",
        "content": "Name: David Lee\nEmail: david@example.com"
      }
    ]
  }'
```

```javascript JavaScript
const response = await fetch('https://alfred-acme.tailnet.ts.net:3100/api/v1/vault/inbox/bulk', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    files: [
      {
        filename: 'meeting-notes-2026-02-26.md',
        content: '# Standup Notes\n\n- Alice: ML pipeline work'
      },
      {
        filename: 'project-idea.md',
        content: '# New Project Idea\n\nBuild a knowledge graph...'
      },
      {
        filename: 'contact-info.txt',
        content: 'Name: David Lee\nEmail: david@example.com'
      }
    ]
  })
});

const result = await response.json();
console.log(result);
```

</RequestExample>

## Response Example

<ResponseExample>

```json 201 Created
{
  "uploaded": 3,
  "files": [
    "inbox/meeting-notes-2026-02-26.md",
    "inbox/project-idea.md",
    "inbox/contact-info.txt"
  ]
}
```

```json 207 Multi-Status (Partial Success)
{
  "uploaded": 2,
  "files": [
    "inbox/meeting-notes-2026-02-26.md",
    "inbox/project-idea.md"
  ],
  "errors": [
    {
      "filename": "contact-info.txt",
      "error": "File already exists: inbox/contact-info.txt"
    }
  ]
}
```

```json 400 Bad Request
{
  "error": "Missing required field: files"
}
```

```json 400 Bad Request
{
  "error": "files must be a non-empty array"
}
```

</ResponseExample>
