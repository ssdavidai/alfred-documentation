---
title: "List Inbox Files"
description: "List all files currently waiting in your inbox."
api: "GET /api/v1/vault/inbox"
---

## Overview

Returns a list of all files currently in your inbox waiting to be processed by the Curator agent. Once the Curator processes a file, it is removed from this list.

## Response

<ResponseField name="files" type="array">
  Array of filenames in the inbox.
</ResponseField>

## Request Example

<RequestExample>

```bash cURL
curl -X GET /api/v1/vault/inbox \
  -H "Authorization: Bearer alf_your_key_here"
```

```javascript JavaScript
const response = await fetch('/api/v1/vault/inbox', {
  method: 'GET',
  headers: {
    'Authorization': 'Bearer alf_your_key_here'
  }
});

const inbox = await response.json();
console.log(inbox);
```

</RequestExample>

## Response Example

<ResponseExample>

```json 200 OK
{
  "files": [
    "meeting-notes-2026-02-26.md",
    "project-idea.md",
    "contact-info.txt"
  ]
}
```

```json 200 OK (Empty Inbox)
{
  "files": []
}
```

</ResponseExample>
