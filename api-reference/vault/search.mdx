---
title: "Search Vault"
description: "Search vault records by keyword or file pattern."
api: "GET /api/v1/vault/search"
---

## Overview

Search your vault records by keyword (grep) or file pattern (glob). Returns matching records or file paths.

## Query Parameters

<ParamField query="glob" type="string" optional>
  File glob pattern to match (e.g., `person/*.md`, `**/*alice*.md`).
</ParamField>

<ParamField query="grep" type="string" optional>
  Text to search for in record content (case-insensitive).
</ParamField>

## Response

<ResponseField name="raw" type="string">
  Search results as text. If the output is structured JSON, it is returned directly instead.
</ResponseField>

## Request Example

<RequestExample>

```bash cURL
curl -X GET \
  "/api/v1/vault/search?grep=machine%20learning&glob=project/*.md" \
  -H "Authorization: Bearer alf_your_key_here"
```

```javascript JavaScript
const params = new URLSearchParams({
  grep: 'machine learning',
  glob: 'project/*.md'
});

const response = await fetch(`/api/v1/vault/search?${params}`, {
  method: 'GET',
  headers: {
    'Authorization': 'Bearer alf_your_key_here'
  }
});

const results = await response.json();
console.log(results);
```

</RequestExample>
