---
title: "Delete Record"
description: "Permanently delete a vault record."
api: "DELETE /api/v1/vault/records/{path}"
---

## Overview

Permanently deletes a record from your vault. Connections from other records that referenced the deleted record will become broken links — the Janitor will detect and can fix these.

<Warning>
  This operation is permanent and cannot be undone.
</Warning>

## Path Parameters

<ParamField path="path" type="string" required>
  Relative path to the record (e.g., `person/Alice Chen.md`). URL-encode spaces and special characters.
</ParamField>

## Response

<ResponseField name="raw" type="string">
  Confirmation message. If the output is structured JSON, it is returned directly instead.
</ResponseField>

## Request Example

<RequestExample>

```bash cURL
curl -X DELETE \
  "/api/v1/vault/records/person/Alice%20Chen.md" \
  -H "Authorization: Bearer alf_your_key_here"
```

```javascript JavaScript
const recordPath = 'person/Alice Chen.md';
const encodedPath = encodeURIComponent(recordPath);

const response = await fetch(`/api/v1/vault/records/${encodedPath}`, {
  method: 'DELETE',
  headers: {
    'Authorization': 'Bearer alf_your_key_here'
  }
});

const result = await response.json();
console.log(result);
```

</RequestExample>

## Response Example

<ResponseExample>

```json 200 OK
{
  "raw": "Deleted person/Alice Chen.md"
}
```

```json 500 Internal Server Error
{
  "error": {
    "code": "EXEC_ERROR",
    "message": "Command failed with exit code 1"
  }
}
```

</ResponseExample>
