---
title: "Upload Inbox File"
description: "Upload a single file to the vault inbox"
api: "POST /api/v1/vault/inbox"
---

## Overview

Uploads a single file to the vault's inbox directory. Files in the inbox are processed by Alfred's Curator daemon, which converts raw inputs into structured vault records. The filename is sanitized to remove dangerous characters before writing.

## Body Parameters

<ParamField body="filename" type="string" required>
  Name of the file to create in the inbox (e.g., `meeting-notes-2026-02-26.md`, `contact-info.txt`)
</ParamField>

<ParamField body="content" type="string" required>
  File contents as a string.
</ParamField>

## Response

<ResponseField name="message" type="string">
  Confirmation message (e.g., `"Uploaded meeting-notes-2026-02-26.md to inbox"`)
</ResponseField>

<ResponseField name="filename" type="string">
  The sanitized filename that was written to the inbox
</ResponseField>

## Request Example

<RequestExample>

```bash cURL
curl -X POST \
  https://app.alfred.black/api/v1/vault/inbox \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "filename": "meeting-notes-2026-02-26.md",
    "content": "# Standup Notes\n\n- Alice: Working on ML pipeline refactor\n- Bob: Reviewing PR #234\n- Carol: Debugging production issue"
  }'
```

```javascript JavaScript
const response = await fetch('https://app.alfred.black/api/v1/vault/inbox', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    filename: 'meeting-notes-2026-02-26.md',
    content: '# Standup Notes\n\n- Alice: Working on ML pipeline refactor\n- Bob: Reviewing PR #234\n- Carol: Debugging production issue'
  })
});

const result = await response.json();
console.log(result);
```

</RequestExample>

## Response Example

<ResponseExample>

```json 201 Created
{
  "message": "Uploaded meeting-notes-2026-02-26.md to inbox",
  "filename": "meeting-notes-2026-02-26.md"
}
```

```json 400 Bad Request
{
  "error": {
    "code": "VALIDATION_ERROR",
    "message": "filename and content are required"
  }
}
```

```json 400 Bad Request (invalid filename)
{
  "error": {
    "code": "VALIDATION_ERROR",
    "message": "Invalid filename"
  }
}
```

```json 401 Unauthorized
{
  "error": {
    "code": "UNAUTHORIZED",
    "message": "Unauthorized"
  }
}
```

</ResponseExample>
