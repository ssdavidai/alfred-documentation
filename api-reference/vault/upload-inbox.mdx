---
title: "Upload to Inbox"
description: "Upload a single file to your inbox for processing."
api: "POST /api/v1/vault/inbox"
---

## Overview

Uploads a single file to your inbox. The Curator agent will automatically process it into structured vault records.

## Body Parameters

<ParamField body="filename" type="string" required>
  Name of the file (e.g., `meeting-notes-2026-02-26.md`).
</ParamField>

<ParamField body="content" type="string" required>
  File contents as a string.
</ParamField>

## Response

<ResponseField name="message" type="string">
  Confirmation message.
</ResponseField>

<ResponseField name="filename" type="string">
  The filename that was written to the inbox.
</ResponseField>

## Request Example

<RequestExample>

```bash cURL
curl -X POST /api/v1/vault/inbox \
  -H "Authorization: Bearer alf_your_key_here" \
  -H "Content-Type: application/json" \
  -d '{
    "filename": "meeting-notes-2026-02-26.md",
    "content": "# Standup Notes\n\n- Alice: Working on ML pipeline refactor\n- Bob: Reviewing PR #234\n- Carol: Debugging production issue"
  }'
```

```javascript JavaScript
const response = await fetch('/api/v1/vault/inbox', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer alf_your_key_here',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    filename: 'meeting-notes-2026-02-26.md',
    content: '# Standup Notes\n\n- Alice: Working on ML pipeline refactor\n- Bob: Reviewing PR #234\n- Carol: Debugging production issue'
  })
});

const result = await response.json();
console.log(result);
```

</RequestExample>

## Response Example

<ResponseExample>

```json 201 Created
{
  "message": "Uploaded meeting-notes-2026-02-26.md to inbox",
  "filename": "meeting-notes-2026-02-26.md"
}
```

```json 400 Bad Request
{
  "error": {
    "code": "VALIDATION_ERROR",
    "message": "filename and content are required"
  }
}
```

</ResponseExample>
