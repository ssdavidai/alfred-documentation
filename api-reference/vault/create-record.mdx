---
title: "Create Record"
description: "Create a new vault record."
api: "POST /api/v1/vault/records"
---

## Overview

Creates a new record in your vault with the specified type, name, metadata fields, and content body.

## Body Parameters

<ParamField body="type" type="string" required>
  Record type (e.g., `person`, `project`, `task`, `assumption`). Must be one of the 19 known types.
</ParamField>

<ParamField body="name" type="string" required>
  Display name for the record. This becomes the filename (e.g., `Alice Chen` creates `person/Alice Chen.md`).
</ParamField>

<ParamField body="fields" type="object" optional>
  Key-value pairs for record metadata. For example: `{"status": "active", "org": "[[org/Acme Corp]]"}`.
</ParamField>

<ParamField body="body" type="string" optional>
  Content body for the record (everything after the metadata).
</ParamField>

## Response

Returns status `201 Created`.

<ResponseField name="raw" type="string">
  Confirmation message. If the output is structured JSON, it is returned directly instead.
</ResponseField>

## Request Example

<RequestExample>

```bash cURL
curl -X POST /api/v1/vault/records \
  -H "Authorization: Bearer alf_your_key_here" \
  -H "Content-Type: application/json" \
  -d '{
    "type": "person",
    "name": "Alice Chen",
    "fields": {
      "role": "ML Engineer",
      "org": "[[org/Acme Corp]]",
      "status": "active"
    },
    "body": "## Background\n\nAlice joined Acme in December 2025."
  }'
```

```javascript JavaScript
const response = await fetch('/api/v1/vault/records', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer alf_your_key_here',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    type: 'person',
    name: 'Alice Chen',
    fields: {
      role: 'ML Engineer',
      org: '[[org/Acme Corp]]',
      status: 'active'
    },
    body: '## Background\n\nAlice joined Acme in December 2025.'
  })
});

const result = await response.json();
console.log(result);
```

</RequestExample>
