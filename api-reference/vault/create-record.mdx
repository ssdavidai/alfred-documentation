---
title: "Create Record"
description: "Create a new vault record"
api: "POST /api/v1/vault/records"
---

## Overview

Creates a new record in the vault with the specified type, name, frontmatter fields, and markdown body. The endpoint runs `alfred vault create` and returns the CLI output.

## Body Parameters

<ParamField body="type" type="string" required>
  Record type (e.g., `person`, `project`, `task`, `assumption`). Must be one of the 19 known types.
</ParamField>

<ParamField body="name" type="string" required>
  Display name for the record. This becomes the filename (e.g., `Alice Chen` becomes `person/Alice Chen.md`).
</ParamField>

<ParamField body="fields" type="object" optional>
  Key-value pairs for frontmatter metadata.
</ParamField>

<ParamField body="body" type="string" optional>
  Markdown body content (everything after frontmatter).
</ParamField>

<ParamField body="scope" type="string" optional>
  Operation scope for authorization (`curator`, `janitor`, `distiller`). Defaults to no scope restrictions.
</ParamField>

## Response

Returns status `201 Created`. The response structure depends on the `alfred vault create` CLI output. If the CLI returns valid JSON, it is passed through directly. Otherwise, the raw text output is wrapped in a `{ "raw": "..." }` object.

<ResponseField name="raw" type="string">
  If the CLI output is not valid JSON, the raw text output is returned in this field.
</ResponseField>

## Request Example

<RequestExample>

```bash cURL
curl -X POST \
  https://app.alfred.black/api/v1/vault/records \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "type": "person",
    "name": "Alice Chen",
    "fields": {
      "role": "ML Engineer",
      "org": "[[org/Acme Corp]]",
      "status": "active"
    },
    "body": "## Background\n\nAlice joined Acme in December 2025."
  }'
```

```javascript JavaScript
const response = await fetch('https://app.alfred.black/api/v1/vault/records', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    type: 'person',
    name: 'Alice Chen',
    fields: {
      role: 'ML Engineer',
      org: '[[org/Acme Corp]]',
      status: 'active'
    },
    body: '## Background\n\nAlice joined Acme in December 2025.'
  })
});

const result = await response.json();
console.log(result);
```

</RequestExample>
