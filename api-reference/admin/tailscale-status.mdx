---
title: "Get Tailscale Status"
description: "Retrieve Tailscale connection and peer information"
api: "GET /api/v1/admin/tailscale/status"
---

Get the current Tailscale status, including information about this node and connected peers. Returns the output of `tailscale status --json`.

<RequestExample>

```bash curl
curl -H "X-API-Key: YOUR_API_KEY" \
  https://alfred-acme.tailnet.ts.net:3100/api/v1/admin/tailscale/status
```

</RequestExample>

<Note>
  The Tailscale status endpoint is useful for verifying connectivity, checking which devices are online, and retrieving the node's Tailscale IP addresses and DNS name.
</Note>
