---
title: "List Backup Snapshots"
description: "List all restic backup snapshots"
api: "GET /api/v1/admin/backups/snapshots"
---

List all backup snapshots stored in the restic repository. This shows the history of all backups taken for this Alfred instance.

<RequestExample>

```bash curl
curl -H "X-API-Key: YOUR_API_KEY" \
  https://alfred-acme.tailnet.ts.net:3100/api/v1/admin/backups/snapshots
```

</RequestExample>

<Note>
  Snapshots are sorted by time (newest first). Automated backups are tagged with "automated", while manual backups (triggered via the API) are tagged with "manual".
</Note>

<Tip>
  To restore from a snapshot, SSH into the instance and use restic directly: `restic -r s3:... restore <snapshot-id> --target /path/to/restore`
</Tip>
