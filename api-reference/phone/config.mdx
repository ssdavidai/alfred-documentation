title: "Phone Config"

api: "GET /api/v1/phone/config"
---

## Overview

Single endpoint backing the dashboard PhonePage. Reads `AGENTPHONE_PHONE_NUMBER` (or alias `TWILIO_PHONE_NUMBER`) from env, the authorised-numbers file, and tails the four phone streams (`voice-call`, `voice-call-outbound`, `sms-inbound`, `sms-outbound`).

## Response

<ResponseField name="phoneNumber" type="string">
  E.164-formatted tenant number, or null if no AgentPhone subnumber is provisioned.
</ResponseField>

<ResponseField name="authorizedNumbers" type="array">
  Phone numbers (normalised) whose inbound SMS gets the openclaw-completion live-reply path.
</ResponseField>

<ResponseField name="recentActivity" type="array">
  Up to 30 events newest-first, merged across the four streams. Each: `{kind: "voice"|"sms", direction: "inbound"|"outbound", from?, to?, summary, at}`.
</ResponseField>

## Request Example

<RequestExample>

```bash cURL
curl -H "Authorization: Bearer $AAS_API_KEY" \
  https://alfred.black/api/v1/phone/config
```

</RequestExample>
