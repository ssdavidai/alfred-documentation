---
title: "Get agent"
description: "Get the resolved model and provider auth for a single agent."
api: "GET /api/v1/admin/agents/{agentId}"
---

Returns the full model status for one agent. For OpenClaw agents (`main`, `learn-clerk`, `vault-curator`, `vault-janitor`, `vault-distiller`) this reads `openclaw.json` for the configured model and shells into the OpenClaw container for live provider-credential status. For `surveyor`, it reads the surveyor block from `config.yaml`.

## Path parameters

<ParamField path="agentId" type="string" required>
  One of `main`, `learn-clerk`, `vault-curator`, `vault-janitor`, `vault-distiller`, `surveyor`.
</ParamField>

## Response â€” OpenClaw agent

<ResponseField name="id" type="string">Agent id.</ResponseField>
<ResponseField name="label" type="string">Display label.</ResponseField>
<ResponseField name="description" type="string">One-line role description.</ResponseField>
<ResponseField name="defaultModel" type="string | null">Model explicitly assigned to this agent in `openclaw.json`, or the global default if no per-agent override is set.</ResponseField>
<ResponseField name="resolvedDefault" type="string | null">Model that will actually be used.</ResponseField>
<ResponseField name="fallbacks" type="string[]">Fallback model chain. Currently always empty in this response.</ResponseField>
<ResponseField name="providers" type="array">
  Provider auth status from `openclaw models status`.
  <Expandable title="Provider fields">
    <ResponseField name="provider" type="string">e.g. `openrouter`, `anthropic`, `google`, `xai`.</ResponseField>
    <ResponseField name="status"   type="string">Whether the credential is configured.</ResponseField>
    <ResponseField name="source"   type="string">Where the credential was found (`env`, `config`).</ResponseField>
  </Expandable>
</ResponseField>
<ResponseField name="missingProviders" type="string[]">Providers required by the configured model but not currently set.</ResponseField>

## Response â€” Surveyor

<ResponseField name="labeler_model" type="string">Cluster-labelling model.</ResponseField>
<ResponseField name="embedder_model" type="string">Embedding model.</ResponseField>
<ResponseField name="embedder_source" type="string">`ollama` or `openrouter`.</ResponseField>
