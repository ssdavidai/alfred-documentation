---
title: "Pause Chore"
description: "Pause a chore — stops the Temporal schedule, sets status=paused."
api: "POST /api/v1/chores/:slug/pause"
---

## Overview

Runs `temporal schedule toggle --pause` against `chore-<slug>` and writes `status: paused` into the chore record. Idempotent.

## Path Parameters

<ParamField path="slug" type="string" required>
  Chore slug.
</ParamField>

## Response

<ResponseField name="slug" type="string">Echo.</ResponseField>
<ResponseField name="status" type="string">`paused`.</ResponseField>

## Request Example

<RequestExample>

```bash cURL
curl -X POST -H "Authorization: Bearer $AAS_API_KEY" \
  https://alfred.black/api/v1/chores/weekly-cashflow-forecast/pause
```

</RequestExample>

## Response Example

<ResponseExample>

```json 200 OK
{ "slug": "weekly-cashflow-forecast", "status": "paused" }
```

```json 404 Not Found
{ "error": "chore weekly-cashflow-forecast not found" }
```

</ResponseExample>
