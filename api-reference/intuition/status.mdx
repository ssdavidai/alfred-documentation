## Overview

Single dashboard endpoint that aggregates everything about the learning subsystem (observations, instincts, reflections), the routing queue, the activity log, and the most recent daily digest. Reads `ALFRED_LEARN_ENABLED` from `/opt/alfred/compose/.env`.

Instinct counts are read from BOTH the legacy `intuition/instincts/` path AND the newer canonical `instinct/` path. This dual-read is a back-compat measure (no data migration was performed when records moved to `instinct/`); it may be removed once legacy records are reconciled.

## Response

<ResponseField name="enabled" type="boolean">Whether the alfred-learn worker is enabled in `.env`.</ResponseField>

<ResponseField name="observations" type="object">
  `{total, unprocessed, processed, invalid}`.
</ResponseField>

<ResponseField name="instincts" type="object">
  `{total, active, proposed, deprecated, merged}`.
</ResponseField>

<ResponseField name="reflections" type="object">`{total}`.</ResponseField>

<ResponseField name="streams" type="object">`{processed}` — count of processed openclaw-session events.</ResponseField>

<ResponseField name="processedToday" type="number">Stream events folded into the vault today (UTC).</ResponseField>

<ResponseField name="autoRouteRate" type="number">Percentage of observations marked `routed_by: alfred`.</ResponseField>

<ResponseField name="queueSize" type="number">Vault records with `status: unrouted`.</ResponseField>

<ResponseField name="recentActivity" type="array">Last 20 lines from `learn-activity.jsonl`, newest-first.</ResponseField>

<ResponseField name="instinctCount" type="number">Total instincts (alias for `instincts.total`).</ResponseField>

<ResponseField name="lastDigest" type="object">
  `{timestamp, path, summary}` — most recent event tagged `digest`. Null if none.
</ResponseField>
