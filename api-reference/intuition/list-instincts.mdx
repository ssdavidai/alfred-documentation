title: "List Instincts"

api: "GET /api/v1/learning/instincts"
---

## Overview

Reads from BOTH `vault/intuition/instincts/` (legacy) and `vault/instinct/` (current). Sorted by name. The `execution` block on each record drives the trust-gradient gate (Tier 1 forced-approval, Tier 3 autonomous) when JudgmentWorkflow fires.

## Response

<ResponseField name="items" type="array">Each instinct record: `{path, name, frontmatter, body, status, created}`. Frontmatter includes the `execution` block (`enabled`, `task_title_template`, `tier`, `requires_approval`).</ResponseField>
<ResponseField name="total" type="number">Count.</ResponseField>

## Request Example

<RequestExample>

```bash cURL
curl -H "Authorization: Bearer $AAS_API_KEY" \
  https://alfred.black/api/v1/learning/instincts
```

</RequestExample>
