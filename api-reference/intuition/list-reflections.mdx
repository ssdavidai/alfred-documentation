## Overview

Reads `vault/reflection/` newest-first. Reflections are the daily 2am Synthesis pass that folds the day's observations into higher-level patterns.

## Response

<ResponseField name="items" type="array">Each: `{path, name, frontmatter, body, status, created}`.</ResponseField>
<ResponseField name="total" type="number">Count.</ResponseField>

## Request Example

<RequestExample>

```bash cURL
curl -H "Authorization: Bearer $AAS_API_KEY" \
  https://alfred.black/api/v1/learning/reflections
```

</RequestExample>
