title: "List Sessions"

api: "GET /api/v1/learning/sessions"
---

## Overview

Reads `vault/session/` newest-first. Sessions are the rolled-up summaries of openclaw conversations across channels (Slack, dashboard, email, voice, SMS).

## Response

<ResponseField name="items" type="array">Each: `{path, name, frontmatter, body, status, created}`.</ResponseField>
<ResponseField name="total" type="number">Count.</ResponseField>

## Request Example

<RequestExample>

```bash cURL
curl -H "Authorization: Bearer $AAS_API_KEY" \
  https://alfred.black/api/v1/learning/sessions
```

</RequestExample>
