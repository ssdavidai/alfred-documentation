## Overview

Lists records from `vault/observation/` newest-first.

## Query Parameters

<ParamField query="limit" type="number" optional default="20">Page size (max 200).</ParamField>
<ParamField query="offset" type="number" optional default="0">Records to skip.</ParamField>

## Response

<ResponseField name="items" type="array">Each record: `{path, name, frontmatter, body, status, created}`.</ResponseField>
<ResponseField name="total" type="number">Total observations across the directory.</ResponseField>
<ResponseField name="limit" type="number">Echo.</ResponseField>
<ResponseField name="offset" type="number">Echo.</ResponseField>

## Request Example

<RequestExample>

```bash cURL
curl -H "Authorization: Bearer $AAS_API_KEY" \
  "https://alfred.black/api/v1/learning/observations?limit=50&offset=0"
```

</RequestExample>
