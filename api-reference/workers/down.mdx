---
title: "Stop Workers"
description: "Stop all Alfred daemon workers"
api: "POST /api/v1/workers/down"
---

Gracefully stops all running Alfred daemon workers by running `alfred down` inside the container.

<RequestExample>

```bash cURL
curl -X POST https://app.alfred.black/api/v1/workers/down \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json"
```

</RequestExample>

<ResponseExample>

```json Response
{
  "message": "Workers stopped"
}
```

</ResponseExample>

<ResponseField name="message" type="string">
  Human-readable status message. Contains the stdout from `alfred down`, or `"Workers stopped"` if stdout is empty.
</ResponseField>
