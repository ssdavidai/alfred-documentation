---
title: "Stop Workers"
description: "Stop all AI agents"
api: "POST /api/v1/workers/down"
---

Gracefully stops all running AI agents. Any in-progress work will be completed before the agents shut down.

<RequestExample>

```bash cURL
curl -X POST /api/v1/workers/down \
  -H "Authorization: Bearer alf_your_key_here" \
  -H "Content-Type: application/json"
```

</RequestExample>

<ResponseExample>

```json Response
{
  "message": "Workers stopped"
}
```

</ResponseExample>

<ResponseField name="message" type="string">
  Confirmation that all agents have been stopped.
</ResponseField>
