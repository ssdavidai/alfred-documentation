---
title: "Get Sweep History"
description: "Retrieve janitor sweep history"
api: "GET /api/v1/workers/janitor/history"
---

Returns a chronological history of janitor sweeps by running `alfred janitor history` inside the container.

<ParamField query="limit" type="number" default={10}>
  Maximum number of sweep records to return. Passed as `--limit` to the underlying command.
</ParamField>

<RequestExample>

```bash Default Limit
curl -X GET https://app.alfred.black/api/v1/workers/janitor/history \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```bash Custom Limit
curl -X GET "https://app.alfred.black/api/v1/workers/janitor/history?limit=25" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

</RequestExample>

<ResponseExample>

```json Structured Response (if alfred outputs JSON)
{
  "...": "Structure depends on alfred janitor history output"
}
```

```json Raw Response (if alfred outputs text)
{
  "raw": "Sweep history (last 10):\n..."
}
```

</ResponseExample>

<Note>
  The response format depends on the `alfred janitor history` command output. If the command outputs valid JSON, it is returned as-is. Otherwise, the raw text output is returned in a `raw` field.
</Note>

<ResponseField name="raw" type="string">
  Raw text output from `alfred janitor history` when the output is not valid JSON. Only present when structured JSON parsing fails.
</ResponseField>
