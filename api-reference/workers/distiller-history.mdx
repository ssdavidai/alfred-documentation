---
title: "Get Extraction History"
description: "Retrieve distiller extraction run history"
api: "GET /api/v1/workers/distiller/history"
---

Returns a chronological history of distiller extraction runs.

<ParamField query="limit" type="number" default={10}>
  Maximum number of run records to return.
</ParamField>

<RequestExample>

```bash Default Limit
curl -X GET https://app.alfred.black/api/v1/workers/distiller/history \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```bash Custom Limit
curl -X GET "https://app.alfred.black/api/v1/workers/distiller/history?limit=25" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

</RequestExample>

<ResponseExample>

```json Structured Response (if alfred outputs JSON)
{
  "...": "Structure depends on alfred distiller history output"
}
```

```json Raw Response (if alfred outputs text)
{
  "raw": "Extraction history (last 10):\n..."
}
```

</ResponseExample>

<Note>
  The response format depends on the `alfred distiller history` command output. If the command outputs valid JSON, it is returned as-is. Otherwise, the raw text output is returned in a `raw` field.
</Note>

<ResponseField name="raw" type="string">
  Raw text output from `alfred distiller history` when the output is not valid JSON. Only present when structured JSON parsing fails.
</ResponseField>
