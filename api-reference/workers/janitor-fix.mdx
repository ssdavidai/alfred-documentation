---
title: "Fix Issues"
description: "Scan vault and fix detected issues using the agent"
api: "POST /api/v1/workers/janitor/fix"
---

Performs a structural scan followed by agent-based automated fixing. The janitor agent will attempt to resolve all detected issues by editing vault files, updating frontmatter, fixing links, and removing orphans.

<Warning>
  This operation modifies vault files. All changes are logged in the mutation audit log, but consider backing up your vault first.
</Warning>

<RequestExample>

```bash cURL
curl -X POST https://app.alfred.black/api/v1/workers/janitor/fix \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json"
```

</RequestExample>

<ResponseExample>

```json Structured Response (if alfred outputs JSON)
{
  "...": "Structure depends on alfred janitor fix output"
}
```

```json Raw Response (if alfred outputs text)
{
  "raw": "Scanned 423 files, found 12 issues, fixed 10"
}
```

</ResponseExample>

<Note>
  The response format depends on the `alfred janitor fix` command output. If the command outputs valid JSON, it is returned as-is. Otherwise, the raw text output is returned in a `raw` field.
</Note>

<ResponseField name="raw" type="string">
  Raw text output from `alfred janitor fix` when the output is not valid JSON. Only present when structured JSON parsing fails.
</ResponseField>
