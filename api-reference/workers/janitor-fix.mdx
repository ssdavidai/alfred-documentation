---
title: "Fix Issues"
description: "Scan vault and fix detected issues using the agent"
api: "POST /api/v1/workers/janitor/fix"
---

Performs a structural scan followed by agent-based automated fixing. The janitor agent will attempt to resolve all detected issues by editing vault files, updating frontmatter, fixing links, and removing orphans.

<Warning>
  This operation modifies vault files. All changes are logged in the mutation audit log, but consider backing up your vault first.
</Warning>

<RequestExample>

```bash cURL
curl -X POST https://alfred-acme.tailnet.ts.net:3100/api/v1/workers/janitor/fix \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json"
```

</RequestExample>
