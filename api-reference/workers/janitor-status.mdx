---
title: "Get Janitor Status"
description: "Get Janitor agent status"
api: "GET /api/v1/workers/janitor/status"
---

Returns the current status of the Janitor agent, including whether it is running and details about its most recent vault health scan.

<RequestExample>

```bash cURL
curl -X GET /api/v1/workers/janitor/status \
  -H "Authorization: Bearer alf_your_key_here"
```

</RequestExample>

<ResponseExample>

```json Structured Response
{
  "status": "running",
  "last_sweep": "2026-02-26T10:15:00Z",
  "issues_found": 12
}
```

```json Raw Response
{
  "raw": "Janitor: running\nLast sweep: 2026-02-26 10:15 (12 issues found)"
}
```

</ResponseExample>

<Note>
  If the response contains structured JSON, it is returned directly. Otherwise, the output is returned in a `raw` field.
</Note>

<ResponseField name="raw" type="string">
  Human-readable output when structured JSON is not available.
</ResponseField>
