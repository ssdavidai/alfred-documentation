---
title: "Get Janitor Status"
description: "Retrieve janitor sweep status and statistics"
api: "GET /api/v1/workers/janitor/status"
---

Returns the current status of the janitor daemon by running `alfred janitor status` inside the container.

<RequestExample>

```bash cURL
curl -X GET https://app.alfred.black/api/v1/workers/janitor/status \
  -H "Authorization: Bearer YOUR_API_KEY"
```

</RequestExample>

<ResponseExample>

```json Structured Response (if alfred outputs JSON)
{
  "...": "Structure depends on alfred janitor status output"
}
```

```json Raw Response (if alfred outputs text)
{
  "raw": "Janitor daemon: running\nLast sweep: 2026-02-26 10:15 (12 issues found)"
}
```

</ResponseExample>

<Note>
  The response format depends on the `alfred janitor status` command output. If the command outputs valid JSON, it is returned as-is. Otherwise, the raw text output is returned in a `raw` field.
</Note>

<ResponseField name="raw" type="string">
  Raw text output from `alfred janitor status` when the output is not valid JSON. Only present when structured JSON parsing fails.
</ResponseField>
