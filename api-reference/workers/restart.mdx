---
title: "Restart Container"
description: "Restart the Alfred Docker container"
api: "POST /api/v1/workers/restart"
---

Restarts the entire Alfred Docker container using `docker compose restart alfred`. This is a hard restart that stops all workers, terminates the container, and starts it fresh. Use this for applying configuration changes or recovering from severe errors.

<Info>
  This operation will cause brief downtime (typically 10-30 seconds) while the container restarts.
</Info>

<RequestExample>

```bash cURL
curl -X POST https://app.alfred.black/api/v1/workers/restart \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json"
```

</RequestExample>

<ResponseExample>

```json Response
{
  "message": "Alfred container restarted"
}
```

</ResponseExample>

<ResponseField name="message" type="string">
  Confirmation message. Always returns `"Alfred container restarted"` on success.
</ResponseField>
