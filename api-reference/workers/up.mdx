---
title: "Start Workers"
description: "Start all AI agents"
api: "POST /api/v1/workers/up"
---

Starts your AI agents. You can optionally start only specific agents by providing a comma-separated list.

<ParamField body="only" type="string" optional>
  Comma-separated list of agents to start. Valid values: `curator`, `janitor`, `distiller`. If omitted, all configured agents will start.

  Example: `"curator,janitor"`
</ParamField>

<RequestExample>

```bash Start All Agents
curl -X POST /api/v1/workers/up \
  -H "Authorization: Bearer alf_your_key_here" \
  -H "Content-Type: application/json"
```

```bash Start Specific Agents
curl -X POST /api/v1/workers/up \
  -H "Authorization: Bearer alf_your_key_here" \
  -H "Content-Type: application/json" \
  -d '{
    "only": "curator,janitor"
  }'
```

</RequestExample>

<ResponseExample>

```json Response
{
  "message": "Workers started"
}
```

</ResponseExample>

<ResponseField name="message" type="string">
  Confirmation that agents have been started.
</ResponseField>
