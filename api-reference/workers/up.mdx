---
title: "Start Workers"
description: "Start Alfred daemon workers"
api: "POST /api/v1/workers/up"
---

Starts the Alfred daemon workers in foreground mode. Can optionally start only specific workers by providing a comma-separated list.

<ParamField body="only" type="string" optional>
  Comma-separated list of workers to start. Valid values: `curator`, `janitor`, `distiller`, `surveyor`. If omitted, all configured workers will start.

  Example: `"curator,janitor"`
</ParamField>

<RequestExample>

```bash Start All Workers
curl -X POST https://app.alfred.black/api/v1/workers/up \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json"
```

```bash Start Specific Workers
curl -X POST https://app.alfred.black/api/v1/workers/up \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "only": "curator,janitor"
  }'
```

</RequestExample>

<ResponseExample>

```json Response
{
  "message": "Workers started"
}
```

</ResponseExample>

<ResponseField name="message" type="string">
  Human-readable status message confirming workers have been started.
</ResponseField>
