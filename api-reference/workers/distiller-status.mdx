---
title: "Get Distiller Status"
description: "Get Distiller agent status"
api: "GET /api/v1/workers/distiller/status"
---

Returns the current status of the Distiller agent, including whether it is running and details about its most recent knowledge extraction run.

<RequestExample>

```bash cURL
curl -X GET /api/v1/workers/distiller/status \
  -H "Authorization: Bearer alf_your_key_here"
```

</RequestExample>

<ResponseExample>

```json Structured Response
{
  "status": "running",
  "last_run": "2026-02-26T09:00:00Z",
  "candidates": 8,
  "learnings_extracted": 12
}
```

```json Raw Response
{
  "raw": "Distiller: running\nLast run: 2026-02-26 09:00 (8 candidates, 12 learnings extracted)"
}
```

</ResponseExample>

<Note>
  If the response contains structured JSON, it is returned directly. Otherwise, the output is returned in a `raw` field.
</Note>

<ResponseField name="raw" type="string">
  Human-readable output when structured JSON is not available.
</ResponseField>
