---
title: "Get Distiller Status"
description: "Retrieve distiller extraction status and statistics"
api: "GET /api/v1/workers/distiller/status"
---

Returns the current status of the distiller daemon.

<RequestExample>

```bash cURL
curl -X GET https://app.alfred.black/api/v1/workers/distiller/status \
  -H "Authorization: Bearer YOUR_API_KEY"
```

</RequestExample>

<ResponseExample>

```json Structured Response (if alfred outputs JSON)
{
  "...": "Structure depends on alfred distiller status output"
}
```

```json Raw Response (if alfred outputs text)
{
  "raw": "Distiller daemon: running\nLast run: 2026-02-26 09:00 (8 candidates, 12 learnings extracted)"
}
```

</ResponseExample>

<Note>
  The response format depends on the `alfred distiller status` command output. If the command outputs valid JSON, it is returned as-is. Otherwise, the raw text output is returned in a `raw` field.
</Note>

<ResponseField name="raw" type="string">
  Raw text output from `alfred distiller status` when the output is not valid JSON. Only present when structured JSON parsing fails.
</ResponseField>
